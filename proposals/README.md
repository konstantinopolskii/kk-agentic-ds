# Proposals

Canon-change proposals live here. Two kinds:
- Retros: `<YYYY-MM-DD>-retro.md`
- Initiatives: `<YYYY-MM-DD>-<slug>.md` (e.g. `2026-04-23-patterns-library.md`)

Proposals are NOT canon. After user review, accepted proposals route to `kk-ds-maintainer` which bakes them into `.claude/skills/` + bumps kit version + ships.
